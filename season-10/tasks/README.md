## How to Use

1. **Before each lesson:** Review the task file to understand what you'll build
2. **During lessons:** Complete hands-on activities with instructor guidance
3. **Between lessons:** Work through daily challenges (30 min/day)
4. **End of week:** Submit your mini-project via GitHub
